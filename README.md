# Renovate Config

To use, add this to your `renovate.json` file.

```json
{
  "$schema": "https://docs.renovatebot.com/renovate-schema.json",
  "extends": ["github>mattmess1221/renovate-config"]
}
```

## Features

- Runs updates on the first weekend every month
- Merge UV updates from all managers into a single PR.
